*This Android application enables voice-controlled lighting via integration with an ESP32-based web server.

**Key Features

Users input the local IP address of the ESP32 web server (hosting a connected relay module and bulb).

Voice commands like "on" or "off" trigger HTTP requests to the corresponding endpoints.

The ESP32 receives the signal and toggles the bulb accordingly.

The app provides a seamless, hands-free solution for smart home lighting control.
